# TrollStore Quick Start Guide

## Быстрая установка TrollStore через LARA

### Шаг 1: Запустить эксплойты
1. Открыть LARA
2. Вкладка "Main" → нажать "Run DarkSword"
3. Дождаться успеха (зеленая галочка)
4. Нажать "Sandbox Escape"
5. Дождаться успеха

### Шаг 2: Установить TrollStore
1. Вкладка "TrollStore"
2. Нажать "Install TrollStore"
3. Дождаться завершения загрузки (прогресс 100%)
4. Открыть приложение "Файлы" (Files)
5. "На iPhone" → "lara" → TrollStore.ipa
6. Нажать на TrollStore.ipa для установки

### Шаг 3: Использование
- TrollStore установлен и готов к работе
- Можно устанавливать любые IPA через TrollStore
- После перезагрузки нужно снова запустить LARA для новых установок

## Требования
- iPhone X или новее
- iOS 17.0 - 17.6.1
- Интернет для загрузки TrollStore

## Что делает LARA
1. **AMFI Bypass** - отключает проверку подписи кода
2. **installd Patch** - патчит системный демон установки
3. **Download TrollStore** - скачивает TrollStore IPA

## Важно
- Патчи действуют до перезагрузки
- После reboot нужно снова запустить LARA
- Это НЕ постоянный jailbreak
- Безопасно для устройства

## Логи
Все логи сохраняются в:
- Файлы → На iPhone → lara → trollstore_logs.txt

## Проблемы?

**"DarkSword не запускается"**
→ Перезагрузить устройство и попробовать снова

**"Sandbox escape failed"**
→ Перезагрузить и повторить

**"Загрузка не начинается"**
→ Проверить интернет-соединение

**"Установка не работает"**
→ Проверить логи в приложении
